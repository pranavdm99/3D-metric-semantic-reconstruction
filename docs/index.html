3D Metric-Semantic Scene Reconstruction

An end-to-end computer vision pipeline converting raw RGB video and ARKit tracking into queryable mathematical 3D scene graphs via Semantic Loop Closure.

Abstract & Fundamental Theory

The reconstruction of 3D environments with semantic understanding is a critical step for autonomous robotics and spatial AI. Traditional SLAM provides sparse geometric tracking but lacks dense surface or semantic object awareness.

This project bridges the gap by integrating 3D Gaussian Splatting (3DGS & SuGaR) for high-fidelity continuous surface reconstruction, and open-vocabulary 2D foundation models (YOLO World and SAM 2) for zero-shot temporal object tracking.

Input Dataset: iPhone 14 RGB (5 FPS)

Phase 1: 3D Surface Reconstruction

The pipeline begins by mathematically estimating camera poses and extracting sparse structural features from the raw video sequence using COLMAP Structure-from-Motion (SFM).

To achieve a continuous, high-fidelity physical geometry, we train a 3D Gaussian Splatting model via SuGaR, generating a dense, mesh-aligned 3D point cloud representing the raw surfaces of the room without any semantic injection.

Phase 2: Semantic Loop Closure

Independently from the 3D splatting, we utilize YOLO World and SAM 2 to perform open-vocabulary zero-shot segmentation and temporal mask tracking across the 2D frame sequence.

These 2D target masks are spatially projected and lifted into the 3D point cloud space. Because object tracking frequently shatters across occlusions (e.g., bed_1 vs bed_2), we introduced a novel Semantic Loop Closure algorithm. It autonomously fuses fragmented point clouds of identical classes based on strict 3D Euclidean bounding box proximity, resulting in clean, singular semantic entities.

Metric Scale & Scene Graphs

Standard COLMAP reconstructions are strictly dimensionless. By extracting the hardware iOS ARKit trajectory alongside the video, we rigidly align the two flight paths using a Sim3 transformation matrix. This mathematically scales the 3D scene directly into real-world physical meters.

We subsequently employ Principal Component Analysis (PCA) strictly against the physical gravity-aligned XZ floor plane. This produces accurate Oriented Bounding Boxes (OBBs) for all semantic objects sans skew distortion. We then compute logical interaction predicates (e.g. resting_on, next_to) to output the final Topological Scene Graph.

Architectural Decisions

Our core framework decision decouple structural visual reconstruction (Phase 1) from open-vocabulary semantic tracking (Phase 2) enabled both layers to execute without encountering localized VRAM memory-overflows.

Discussion & LLM Querying

Rather than injecting millions of highly dense, texturally colored 3D splats into a slow vision-transformer, we compile the topological Scene Graph into an ultra-lightweight rigid JSON structure.

When injected as context into Ollama (Llava/Llama-3), the agent possesses pure, hallucination-free spatial awareness. It organically reasons about physical metric distance and structural geometry in single-shot inference.

Limitations & Future Work

Relying on SAM 2 temporal masks inherently struggles against rapidly moving, deformable subjects due to extreme motion blur before 3D lifting occurs. Future permutations will directly query visual clip-embedding vectors layered directly into the Gaussian Splats, bypassing 2D temporal occlusion tracking entirely.
